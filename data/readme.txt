Training data

The training-data.tar.gz contains the following:
1. train-images - Training images (jpg files)
2. train.csv has two comma separated columns. First is the image name and the second one is the numeric class
3. class-mapping-num-jp-en.csv - The file contains the mapping from the numeric class to the class names in Japanese and English. It has three comma-separated columns: numeric class, class name in Japanese, and the class name translated to English using Google Translate
